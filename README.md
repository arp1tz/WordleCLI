# WordleCLI
Wordle is word guessing game in which we have to guess a 5-Letter word in a limited tries, I am trying to create a wordle based in Terminal which will be in Command Line Interface which sounds fun.

A terminal-based Wordle clone built using Node.js.

This project recreates the classic Wordle experience directly inside the terminal with a reactive keyboard, colorful feedback system, replay functionality, and clean CLI aesthetics.

---

## Features

-  ASCII Art Title
-  Reactive Keyboard UI
-  Word Feedback System
-  Dictionary Validation
-  Replay Functionality
-  Random Secret Word Selection
-  Guess History Tracking
-  Clean Terminal Interface
